CmdSwitcher

Commandキーで、瞬時に入力切り替え

MacのUS配列キーボードで
左Command → 英数、右Command → かな
シンプル、軽量、誤動作なし

無料ダウンロード v1.2.0 / macOS 12+

特徴

⚡

超軽量

わずか172KB。メモリ使用量も最小限。バックグラウンドで動作してもPCに負荷をかけません。

🎯

誤動作なし

Command+Cなどのショートカット時は発動しません。単独で短く押した時だけ切り替わります。

💻

ユニバーサル対応

Intel MacとApple Silicon Macの両方に対応したユニバーサルバイナリ。

使い方

左 ⌘ Command
→
英数入力
右 ⌘ Command
→
かな入力

※ 300ms以上の長押しや、他のキーとの組み合わせでは発動しません

なぜCmdSwitcher？

⌘英かななど既存のツールは、macOSのアップデートで動作しなくなったり、 スペースキーやエンターキーが反応しなくなる不具合が報告されています。

CmdSwitcherはシンプルな実装と誤動作防止ロジックで、 安定した動作を実現しています。
